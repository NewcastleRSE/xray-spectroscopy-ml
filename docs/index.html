X A N E S N E T


XANESNET is an open source project developed through the Penfold Group @ Newcastle University which provides machine learning tools and techniques for the fast and automated analysis and prediction of X-ray spectroscopy data. This documentation provides a guide to interacting with and contributing to XANESNET.
